# Week 2 Demo Project: CSS Fundamentals (Portfolio Styling)

This demo project helps students understand:
- CSS selectors (element, class, id, descendant)
- Box model (padding, border, margin, shadow)
- Typography and colors
- Positioning (static, relative, absolute, fixed)
- Workshop: Styling the Week 1 portfolio

## How to Run
1. Extract the ZIP.
2. Open `index.html` in your browser.
3. Edit `style.css` and refresh the browser to see changes.

## Files
- `index.html` (About + skills + positioning demo)
- `projects.html` (Card grid)
- `contact.html` (Styled form)
- `style.css` (All CSS fundamentals)
